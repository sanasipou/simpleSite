# 🌐 Mon Premier Site Web

Un site web simple créé avec **HTML5** et **CSS3** pour apprendre les bases du développement web.

## 📂 Structure du projet

```text
mon-projet/
│
├── index.html
├── style.css
└── README.md
```

## 🚀 Fonctionnalités

* En-tête avec menu de navigation
* Section de présentation (Hero Section)
* Bouton d'action
* Cartes de présentation des technologies
* Pied de page
* Design responsive simple

## 🛠️ Technologies utilisées

* HTML5
* CSS3

## 📋 Installation

1. Télécharger ou cloner le projet.
2. Ouvrir le dossier du projet.
3. Lancer le fichier `index.html` dans un navigateur web.

## 🎨 Aperçu

Le site contient :

* Une barre de navigation.
* Une section d'accueil avec un titre et une description.
* Trois cartes présentant HTML, CSS et JavaScript.
* Un pied de page.

## 📚 Objectif pédagogique

Ce projet permet d'apprendre :

* La structure d'une page HTML.
* L'utilisation des balises sémantiques.
* La mise en forme avec CSS.
* L'utilisation de Flexbox.
* Les effets de survol (hover).

Projet réalisé dans le cadre de l'apprentissage du développement web.
